# Алгоритмы: сортировка и поиск

Практическая реализация базовых алгоритмов сортировки и поиска на Python  
с анализом их производительности.

## Содержимое

### Сортировка
- `bubble_sort_array.py` — пузырьковая сортировка массива
- `bubble_sort_matrix.py` — сортировка строк и столбцов матрицы
- `bubble_sort_external.py` — внешняя сортировка (работа с файлами)

- `gnome_sort_array.py` — гномья сортировка массива
- `gnome_sort_matrix.py` — гномья сортировка матрицы
- `gnome_sort_external.py` — внешняя гномья сортировка

### Поиск
- `brute_force_search.py` — поиск полным перебором
- `binary_search.py` — двоичный поиск (для отсортированных данных)

### Дополнительно
- `sorting_searching.pdf` — курсовая работа с теорией и результатами

## Что реализовано

- реализация алгоритмов “с нуля”
- подсчёт:
  - количества сравнений
  - количества перестановок
- измерение времени выполнения
- работа с:
  - массивами
  - матрицами
  - файлами (внешняя сортировка)


## Результаты

- алгоритмы с квадратичной сложностью O(n²) (bubble, gnome)  
  значительно замедляются при увеличении объёма данных  
- двоичный поиск работает быстрее полного перебора,  
  но требует предварительной сортировки данных
