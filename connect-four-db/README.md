# Connect Four (БД + Python)

Проект по разработке игры «Четыре в ряд» с использованием базы данных Oracle и клиентского приложения на Python.

🔗 Репозиторий проекта:  
https://github.com/ddyonth/connect-four-oracle

## Описание

Реализована настольная игра с хранением состояния в базе данных и клиентским интерфейсом.
Логика игры реализована на стороне базы данных, что позволяет централизованно управлять игровым процессом и поддерживать многопользовательский режим.

Архитектура:
- логика игры реализована на уровне БД (PL/SQL)
- клиентская часть реализована на Python (tkinter)

## Функционал

- игра для двух пользователей
- хранение истории партий
- расчет рейтинга игроков
- поддержка нескольких игровых сессий
- настройка размера поля (7×6 до 10×8)
- таймер хода
- автоматическое определение победы/ничьей
- возможность реванша

## База данных

Реализованы таблицы:
- Players — игроки
- Game_history — история игр
- Current_game — активные сессии
- Moves — ходы
- Rating — рейтинг игроков

Основная логика реализована в пакете:

- `start_game` — создание игры  
- `make_move` — выполнение хода  
- `check_winner` — проверка победы  
- `update_rating` — обновление рейтинга  
- `rematch` — повторная игра  

## Клиентское приложение

Python-приложение реализует:
- графический интерфейс (tkinter)
- отображение игрового поля
- взаимодействие с базой данных
- управление ходами и сессиями

## Технологии

Python, Tkinter, Oracle Database, PL/SQL.
