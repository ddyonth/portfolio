# MindSpark — приложение для создания викторин (Qt)

Курсовой проект по разработке десктопного приложения для создания, редактирования и прохождения викторин.

🔗 Репозиторий проекта:  
https://github.com/edersik/MindSpark

## Описание

Приложение разработано на C++ с использованием Qt и представляет собой инструмент для создания и прохождения тестов с графическим интерфейсом.

Поддерживается:
- создание викторин
- редактирование вопросов
- загрузка и сохранение тестов (JSON)
- прохождение викторин с таймером
- таблица результатов

## Архитектура

Приложение построено на основе модульной структуры:

- MainWindow — главное окно  
- QuizEditor — создание и редактирование тестов  
- QuizTaker — прохождение викторины  
- QuizViewer — просмотр викторин  

Взаимодействие компонентов реализовано через механизм сигналов и слотов Qt.

## Особенности реализации

- хранение данных в JSON  
- динамический расчет времени прохождения  
- система оценки сложности вопросов  
- фильтрация результатов по викторинам  
- обработка пользовательского ввода и валидация данных
