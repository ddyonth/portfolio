# Agent System for Process Monitoring (НИР)

Научно-исследовательский проект по разработке агентной системы мониторинга процессов для анализа поведения и выявления отклонений от накопленной истории активности.

🔗 Репозиторий проекта:  
https://github.com/ddyonth/monitoring-project

## Описание

Проект представляет собой систему мониторинга на базе агентной архитектуры. В основе исследования лежит метод выявления аномалий путем сравнения текущих действий процессов с предварительно накопленной историей их нормальной активности. Система фиксирует события и оповещает об отклонениях, не используя при этом предопределенные признаки вредоносности, а опираясь на уникальный профиль поведения каждой среды.

Работа выполнена в рамках НИР и направлена на создание надежного инструмента поведенческого аудита в операционных системах.

## Функционал

- **Сбор системных событий**: Непрерывная фиксация активности процессов (файловые операции, системные вызовы).
- **Накопление базы профилей**: Формирование «белого списка» или эталонной истории поведения для каждого процесса.
- **Обнаружение отклонений**: Выявление действий, которые ранее не фиксировались в истории активности данного процесса.
- **Агентная архитектура**: Использование легковесных агентов для минимизации нагрузки на наблюдаемую систему.

## Исследовательская часть

В рамках НИР реализованы:
- Механизм структурирования и хранения истории активности процессов.
- Алгоритмы сравнения текущих векторов активности с накопленной базой данных.
- Система фильтрации легитимных изменений, позволяющая снизить уровень ложных срабатываний при обновлении профилей поведения.

## Технологии

Python, SQL (хранение истории), System API Monitoring.
