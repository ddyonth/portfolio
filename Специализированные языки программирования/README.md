# Специализированные языки и технологии программирования (Qt, C++)

Лабораторные работы по разработке приложений на Qt с использованием C++.

В рамках работ:
- разработка клиент-серверных приложений (QtNetwork)
- работа с потоками ввода-вывода (QIODevice)
- реализация многопоточности (QThread)
- использование баз данных в приложениях
- построение архитектуры приложений (MVC)
- разработка пользовательских интерфейсов и кастомных виджетов

## Список работ

- `Комплексный проект (ЛР17).pdf` — комплексное приложение с интеграцией сети, ввода-вывода и пользовательского интерфейса  
- `Сетевое приложение (ЛР10).pdf` — реализация клиент-серверного взаимодействия с использованием QtNetwork  
- `Работа с вводом-выводом (ЛР9).pdf` — работа с потоками данных и файлами через QIODevice  
- `Многопоточность (ЛР8).pdf` — реализация многопоточности с использованием QThread  
- `Работа с БД (ЛР7).pdf` — работа с базой данных в Qt-приложении  
- `Архитектура MVC (ЛР5).pdf` — разработка графического приложения с использованием паттерна MVC  
- `Кастомный виджет (ЛР4).pdf` — создание пользовательского виджета с переопределением отрисовки  

## Используемые технологии

C++, Qt (Qt Widgets, QtNetwork, QThread, QIODevice), MVC.
