# Методы оптимизации

Практические работы по решению задач линейного программирования и реализации алгоритмов оптимизации.

## Содержимое

- `simplex_method.pdf` — реализация симплекс-метода  
- `brute_force_lp.pdf` — решение задачи линейного программирования методом полного перебора с использованием метода Гаусса  

## Что реализовано

- приведение задачи к канонической форме  
- работа с базисными и опорными решениями  
- реализация симплекс-метода  
- программная реализация алгоритмов оптимизации  
- решение систем линейных уравнений (метод Гаусса)  
- перебор допустимых решений и выбор оптимального  

## Используемые технологии

Python, NumPy.
