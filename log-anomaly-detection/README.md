# Log Anomaly Detection (Machine Learning)

Проект по разработке системы автоматического обнаружения аномалий в логах серверов с использованием методов машинного обучения.

🔗 Репозиторий проекта:  
https://github.com/ddyonth/log-anomaly-detection

## Описание

В проекте реализован инструмент для анализа текстовых логов с целью выявления подозрительной активности и системных сбоев. Система преобразует неструктурированные строки логов в числовые векторы и использует алгоритмы машинного обучения для поиска отклонений от нормального паттерна поведения.

Основные этапы:
- Парсинг и предобработка неструктурированных данных
- Векторизация событий (логи в числовые признаки)
- Обучение моделей для выделения "нормального" состояния системы

## Функционал

- Автоматическое извлечение шаблонов из логов (Log Parsing)
- Детекция аномалий на основе статистических методов и ML
- Анализ последовательностей событий во времени
- Визуализация результатов и выявленных инцидентов
- Оценка точности моделей (Precision, Recall, F1)

## Техническая часть

Реализованы компоненты:
- **Data Loader**: загрузка и первичная обработка сырых файлов логов.
- **Preprocessing**: очистка данных, удаление параметров (IP, ID), приведение к шаблонам.
- **Modeling**: использование алгоритмов (например, Isolation Forest, SVM или нейросетевые подходы) для поиска выбросов.
- **Evaluation**: скрипты для проверки качества работы алгоритма на тестовых данных.

## Технологии

Python, Pandas, Scikit-learn, NumPy, Matplotlib.
