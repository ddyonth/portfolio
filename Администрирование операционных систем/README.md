# Администрирование операционных систем

Лабораторные работы по администрированию Linux-систем на примере ОС «Альт Рабочая станция».

В рамках работ:
- управление процессами и системными службами
- администрирование пользователей и прав доступа
- настройка сети и удаленного доступа
- мониторинг и журналирование системы
- использование средств централизованной аутентификации
- автоматизация конфигурации

## Список работ

- `process_management.pdf` — управление процессами в системе  
- `user_group_management.pdf` — управление пользователями и группами  
- `access_control_linux.pdf` — разграничение прав доступа  
- `remote_access_ssh.pdf` — настройка удаленного доступа (SSH)  
- `network_configuration.pdf` — настройка сетевых параметров  
- `journald_logging.pdf` — работа с системным журналом journald  
- `system_monitoring.pdf` — мониторинг и диагностика системы  
- `ansible_configuration.pdf` — автоматизация конфигурации с использованием Ansible  
- `kerberos_authentication.pdf` — настройка службы аутентификации Kerberos  
- `freeipa_domain.pdf` — развертывание доменной инфраструктуры FreeIPA
- `samba_file_server.pdf` — настройка файлового сервера Samba для организации общего доступа к файлам в сети  
- `nfs_file_sharing.pdf` — настройка сетевой файловой системы NFS для совместного использования каталогов между узлами  

## Используемые технологии

Linux, Альт Рабочая станция, SSH, journald, Ansible, Kerberos, FreeIPA.
