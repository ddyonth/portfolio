# Машинное обучение

В данной папке собраны лабораторные работы по дисциплине «Машинное обучение».  
Работы выполнены в формате Jupyter Notebook и включают анализ данных, визуализацию, подготовку признаков, обучение моделей и оценку качества результатов.

## Список работ

- `lr1_linear_regression.ipynb` — исследование датасета о прочности бетона, построение моделей линейной и полиномиальной регрессии, оценка качества предсказаний.
- `lr2_logistic_regression.ipynb` — построение модели логистической регрессии для прогнозирования оттока клиентов.
- `lr3_svm.ipynb` — классификация уровня успеваемости студентов с использованием метода опорных векторов.
- `lr4_decision_tree.ipynb` — классификация назначаемого препарата с помощью модели дерева решений.

## Используемые инструменты

Python, pandas, numpy, matplotlib, seaborn, scikit-learn.
