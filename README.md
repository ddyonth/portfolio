# Portfolio

Репозиторий содержит учебные и проектные работы в области анализа данных, машинного обучения и системной инженерии.

Основной фокус:  
- работа с данными  
- алгоритмы и аналитика  
- построение и исследование моделей  
- системное понимание IT-инфраструктуры  

---

## Ключевые навыки

- Python (pandas, базовый ML)
- SQL / базы данных / PL/SQL
- анализ данных и подготовка данных
- базовая статистика
- алгоритмы и структуры данных
- сетевые технологии и системное администрирование
- работа с логами и мониторингом
