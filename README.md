# Portfolio

Репозиторий содержит учебные работы и личные проекты, выполненные в процессе освоения инструментов анализа данных, машинного обучения и смежных технических дисциплин. Работы демонстрируют навыки применения Python и SQL для решения практических и исследовательских задач.

## Ключевые навыки

* **Анализ данных и ML**: Реализация моделей классификации, регрессии и кластеризации на Python с использованием стандартных библиотек анализа данных.
* **Базы данных**: Проектирование реляционных структур, написание SQL-запросов и логики на стороне СУБД (PL/SQL, Oracle).
* **Программирование**: Разработка инструментов на Python (FastAPI, psutil) и базовый опыт разработки на C++ (Qt).
* **Технический анализ**: Работа с логами, мониторинг системных процессов и низкоуровневый анализ данных.

## Основные проекты

### [Connect Four](https://github.com/ddyonth/connect-four-oracle) (Python + Oracle DB)
Разработка настольной игры с клиент-серверной архитектурой, где основная игровая логика и управление сессиями реализованы на уровне базы данных.
* **Стек**: Python (Tkinter), Oracle Database, PL/SQL.

### [Monitoring Project](https://github.com/ddyonth/monitoring-project)
Система мониторинга процессов на рабочих станциях Windows, предназначенная для сбора телеметрии и выявления отклонений в потреблении ресурсов.
* **Стек**: Python, psutil, FastAPI, SQLite.

### [Log Anomaly Detection](https://github.com/ddyonth/log-anomaly-detection)
Проект по анализу системных журналов и поиску аномалий в поведении процессов на основе накопленной истории.
* **Стек**: Python, Pandas, Scikit-learn.

### [MindSpark Quiz App](https://github.com/edersik/MindSpark)
Командная разработка приложения-викторины с упором на проектирование архитектуры и сценариев взаимодействия.

## Структура репозитория

* **Машинное обучение**: Лабораторные работы по обучению моделей линейной и логистической регрессии, SVM и деревьев решений.
* **Базы данных**: Примеры работы с реляционными БД, написание процедур и триггеров.
* **Алгоритмы**: Реализация алгоритмов поиска и сортировки, анализ их временной эффективности.
* **Методы оптимизации**: Решение математических задач (симплекс-метод, метод Гаусса) на Python.
