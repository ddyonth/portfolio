# Портфолио учебных проектов

Репозиторий содержит выполненные учебные проекты и лабораторные работы по направлениям:
- машинное обучение
- администрирование операционных систем
- сетевые технологии
- разработка приложений (Qt, C++)
- базы данных

## Структура

### Машинное обучение
Практические работы с использованием Python и библиотек машинного обучения (Jupyter Notebook).

---

### Операционные системы (Windows Server)
Администрирование доменной инфраструктуры:
- Active Directory, DNS, DHCP
- групповые политики (GPO)
- разграничение прав доступа
- PKI, IPsec
- централизованное журналирование

---

### Операционные системы (Linux, Альт)
Администрирование Linux-систем:
- управление пользователями и правами доступа
- настройка сети и удаленного доступа
- мониторинг и журналирование
- Ansible, Kerberos, FreeIPA
- сетевые файловые системы (Samba, NFS)

---

### Сети и системы передачи информации
Настройка сетевой инфраструктуры в PNETLab:
- VLAN, QinQ
- STP, SPAN / RSPAN
- статическая и динамическая маршрутизация (OSPF, BGP)
- MPLS
- анализ сетевого трафика (Wireshark)

---

### Разработка приложений (Qt, C++)
Разработка приложений с использованием Qt:
- клиент-серверные приложения (QtNetwork)
- многопоточность (QThread)
- работа с вводом-выводом (QIODevice)
- архитектура приложений (MVC)
- пользовательские интерфейсы

---

### Проект с базой данных
Игра «Четыре в ряд»:
- логика реализована на уровне БД (PL/SQL)
- клиентское приложение на Python (tkinter)
- хранение истории игр и рейтинга игроков

---
### MindSpark — приложение для викторин (Qt)

Курсовой командный проект по разработке десктопного приложения для создания и прохождения викторин.

- проектирование архитектуры приложения (UML, C4)
- разработка UI/UX макетов (Balsamiq)
- описание логики и пользовательских сценариев
- тестирование и валидация функциональности

Репозиторий: https://github.com/edersik/MindSpark

---

## Используемые технологии

Python, C++, Qt, Oracle Database, PL/SQL, Linux, Windows Server, PNETLab, Wireshark.
