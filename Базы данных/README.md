# Работа с базами данных (Oracle, PL/SQL)

Практические работы по разработке логики на стороне базы данных с использованием Oracle и PL/SQL.

Основной акцент сделан на реализации бизнес-логики, обработке данных и использовании встроенных механизмов СУБД.

## Содержимое

- `plsql_logic.sql` — примеры реализации логики на стороне БД:
  - функции (определение типа операции)
  - процедуры с логированием выполнения
  - пакет (package) для управления бронированиями
  - триггер для валидации данных
  - планировщик задач (DBMS_SCHEDULER)
  - материализованные представления для оптимизации запросов

## Что реализовано

- обработка и анализ данных с использованием SQL и регулярных выражений  
- разработка процедур с фиксацией выполнения (логирование)  
- реализация бизнес-логики на уровне базы данных  
- валидация входных данных с помощью триггеров  
- работа с пакетами (package) и курсорами  
- использование планировщика задач Oracle  
- оптимизация запросов через materialized view  

## Используемые технологии

Oracle Database, SQL, PL/SQL.
