# Современные операционные системы

Лабораторные работы по администрированию и обеспечению безопасности Windows.

В рамках работ:
- развертывание доменной инфраструктуры (Active Directory, DNS)
- управление пользователями и правами доступа
- настройка групповых политик (GPO)
- организация сетевых служб (DHCP, IIS, FTP)
- настройка защищённых соединений (IPsec)
- развертывание инфраструктуры открытых ключей (PKI)
- централизованный сбор и анализ журналов событий
- настройка удаленного доступа (RDP, RemoteApp)

## Список работ

- `ad_dns_setup.pdf` — установка и настройка контроллера домена и DNS-сервера  
- `user_management.pdf` — управление доменными пользователями и группами  
- `group_policy_gpo.pdf` — настройка групповых политик (GPO)  
- `access_control_permissions.pdf` — разграничение прав доступа пользователей  
- `pki_deployment.pdf` — развертывание инфраструктуры открытых ключей (PKI)  
- `ipsec_secure_ftp.pdf` — настройка защищённых соединений с использованием IPsec и безопасного FTP  
- `centralized_logging.pdf` — настройка централизованного сбора журналов событий  
- `dhcp_setup.pdf` — настройка DHCP-сервера  
- `iis_ftp_setup.pdf` — установка и базовая настройка IIS и FTP-сервера  
- `remoteapp_rdp.pdf` — настройка удаленного доступа (RemoteApp, RDP)

## Используемые технологии

Windows Server, Active Directory, DNS, DHCP, Group Policy, PKI, IPsec, IIS, FTP, RDP.
